# Inductor Winder — Project Summary

## Reference Frame
- **X axis**: longitudinal (wire feed direction, supply spool to spindle)
- **Y axis**: latitudinal (spindle axis, traversal direction, cross members parallel to this)
- **Z axis**: vertical

---

## Project Overview
Designing a precision inductor winder for 36 AWG magnet wire with:
- Cylindrical barrel cam traversal mechanism
- Active tension control via brushed DC gearmotor on supply spool
- Dancer arm tension sensing with AS5600 magnetic encoder
- NEMA 17 stepper spindle drive
- 4-stage GT2 belt reduction (92.16:1) from spindle to cam
- Simulink control system model (completed and tuned)
- STM32 microcontroller for real hardware implementation
- 4040 extrusion frame

---

## Cam Design
```
Type:              Reversing single-groove barrel cam (continuous zigzag)
                   NOT two separate grooves — eliminates groove crossing interference
Cam radius:        R_cam = 30mm
Stroke:            76mm
Motion law:        Cycloidal
Cam revolutions
per traverse:      5.8811 (rise OR return = one layer)
Groove type:       V-groove, 90° included angle
Follower:          3mm Si3N4 ceramic ball
Groove width:      ~3.5mm (ball + clearance)
```

**Important:** Original two-groove design was abandoned because groove centerlines came within 2.2mm of each other at closest approach — not enough clearance for 3mm ball + walls. Single reversing groove has no crossing issue; adjacent parallel passes have 12.7mm axial spacing.

**Cam CSV files:** generated by Matlab, can import the splines in Fusion using dxfBuildSpline.

## Power Transmission (Belt Drive)
```
Type:         4-stage GT2 timing belt
Total ratio:  92.16:1
Belt width:   6mm

Stage 1: 18T → 72T  =  4.0:1   (belt ~196mm / 98T)
Stage 2: 18T → 72T  =  4.0:1   (belt ~196mm / 98T)
Stage 3: 20T → 48T  =  2.4:1   (belt ~160mm / 80T)
Stage 4: 20T → 48T  =  2.4:1   (belt ~160mm / 80T)

Shafts:       6mm diameter, all parallel to Y axis
Bearings:     KP06 pillow blocks (6mm bore)
Tensioning:   Floating cross members slide in 4040 T-slots
```

**Belt plane staggering (axial, to prevent stage interference):**
- Stages 1 & 3: Plane A (inner)
- Stages 2 & 4: Plane B (outer, +15mm offset)

**Winding geometry with 92.16:1 and 6 cam revs per traverse:**
```
Turns per layer:  92.16 × 6 = 553 (theoretical 542 needed for close wind)
Pitch:            76 / 553 = 0.137mm  (~2% under wire dia — within enamel tolerance)
```

```
Nr (rotor teeth): 50

J_rotor:       68e-7 kg·m²

Drive:         Direct drive to bobbin shaft via 5mm→8mm rigid coupling

```

**Resonant frequency:** ~43 Hz (288 rad/s at motor shaft). At 1/16 microstepping the resonant shaft speed is very low (~0.09 rad/s) — passed through quickly during ramp.

## Bobbin
```
R_core:        4.5mm = 0.0045m
Winding length: 76mm (= cam stroke)
Shaft:         8mm, two KP06 pillow blocks
Drive:         NEMA 17 via 5mm→8mm rigid shaft coupling
```

**Winding geometry:**
```
Turns per layer:   floor(0.076 / 0.140e-3) = 542 (theoretical close wind)
R_eff at layer n:  R_core + (n + 0.5) × wire_diameter
Layer transition:  one cam traverse = one layer
```

## Control System

Modelled in 'winder.slx'

### Spindle Speed Controller
```
Architecture:  Feedforward + PI correction
Feedforward:   step_rate_ff = omega_target × (200 × microstep_divisor) / (2π)
PI correction: step_rate_corr = Kp × error + Ki × integral(error)
Rate limiter:  5.24 rad/s² on omega_target (ramp from 0 to setpoint)
Kp:            1.0 steps/(rad/s)   [tuned empirically]
Ki:            0.2 steps/(rad/s/s) [tuned empirically]
Anti-windup:   ±800 steps/s clamp on integrator output
Notch filter:  On omega_spindle feedback, omega_n ≈ 42 rad/s, zeta=0.3
               num = [1, 0, 1764]
               den = [1, 25.2, 1764]
```

### Tension Controller
```
Architecture:  P controller (plant is integrating, so P alone gives zero SS error
               with integral in plant — add small Ki if drift observed)
Feedback:      T_measured from dancer arm AS5600 (direct physical measurement)
               NOT back-EMF estimator (estimator too slow for transients)
Kp:            90 V/N   [tuned in simulation]
Ki:            165 V/(N·s) [tuned in simulation]
Anti-windup:   ±12V clamp
Sign:          Positive error (T < setpoint) → positive V_cmd → motor drives
               spool forward → more wire paid out → tension rises
Motor is bidirectional — can brake OR drive spool
```

### Simulink Model Structure
All subsystems completed and tuned:
1. **Spindle Speed Controller** — feedforward + PI, rate limiter, notch filter
2. **Stepper Motor Model** — full electromagnetic (iA, iB, omega, theta ODEs)
   - Quasi-static current approximation used for speed (removes stiffness, 10-50x faster)
3. **Stepper Driver** — sinusoidal microstepping, VA=cos(theta_e), VB=sin(theta_e)
   - theta_elec = step_count × 2π / (4 × microstep_divisor)  (NO Nr in denominator)
4. **Cam/Follower Lookup** — Uses same .csv as Fusion, two Constant blocks feed cam_theta + cam_z
   - Precomputed dz/dtheta and d2z/dtheta2 arrays (gradient()) passed as separate inputs
   - Three 1-D Lookup Table blocks replace MATLAB Function for speed
5. **Winding Geometry Tracker** — R_eff, wire_rate, layer_number, turns_total, T_load
   - layer_number = floor(theta_spindle / (2π × gear_ratio))
6. **Disturbance Model** — cam reversal (tension spike) + layer transition (tension drop)
7. **Wire Tension Model** — dT/dt = K_wire × (v_consumed - v_supplied) + disturbance
   - Integrator IC = T_pretension = 0.010N
8. **DC Spool Motor Model** — two transfer functions (electrical + mechanical)
   - T_load referred to motor shaft: T_wire × R_supply / (gear_ratio × eta)
9. **Tension Controller** — PI with anti-windup
10. **Dancer Arm Model** — second-order ODE with K_spring, B_arm, J_arm
    - Output theta → K_sensor → T_measured
11. **Safety/Limit Block** — e-stop at T > 0.080N or T < 0.005N
